Usage:
Prerequisites:
1. Have `gcc` or `clang` installed in `PATH`
2. have `glslc` installed in `PATH`
Prepare build:
1. Run `regen_extern` to clone extern dependencies
2. Adjust compile settings in `regen_cmake`
3. Run `regen_cmake`
Build:
- `relase`
